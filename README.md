MobileShopASD is an advanced player marketplace infrastructure plugin focused on creating immersive, progression-based and highly interactive shop ecosystems.

Unlike traditional player shop plugins, MobileShopASD was designed as a complete commercial infrastructure connecting:
- <img src="https://raw.githubusercontent.com/ItsAsddd/ItsAsddd/main/mcicons/minecraft_emerald.png" width="16"> Player shops
- <img src="https://raw.githubusercontent.com/ItsAsddd/ItsAsddd/main/mcicons/minecraft_barrel.png" width="16"> Physical markets
- <img src="https://raw.githubusercontent.com/ItsAsddd/ItsAsddd/main/mcicons/minecraft_book.png" width="16"> Discord commerce
- <img src="https://raw.githubusercontent.com/ItsAsddd/ItsAsddd/main/mcicons/minecraft_beacon.png" width="16"> Progression systems
- <img src="https://raw.githubusercontent.com/ItsAsddd/ItsAsddd/main/mcicons/minecraft_totem_of_undying.png" width="16"> Ranking infrastructure
- <img src="https://raw.githubusercontent.com/ItsAsddd/ItsAsddd/main/mcicons/minecraft_hopper.png" width="16"> Offline delivery systems

The plugin transforms trading into a real gameplay system instead of simple chest interactions.

### <img src="https://raw.githubusercontent.com/ItsAsddd/ItsAsddd/main/mcicons/minecraft_paper.png" width="18"> Chat Integration

Players can quickly access their shops directly from chat using:
```txt
#shop
```

This creates a much faster and cleaner trading experience, especially for active servers with large economies.

### <img src="https://raw.githubusercontent.com/ItsAsddd/ItsAsddd/main/mcicons/minecraft_redstone.png" width="18"> Administrative Infrastructure

Administrators can:
- manage shop levels
- edit market instances
- force upgrades
- synchronize NPCs
- configure ratings
- control cooldowns
- manage boosters
- synchronize Discord systems

without requiring server restarts.

### <span style="color:#F6B04E"><b>Marketplace-Oriented Design</b></span>

The objective was creating a scalable player economy ecosystem rather than isolated player shops.
